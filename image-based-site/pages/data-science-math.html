Understanding mathematical functions is crucial for data scientists, as they help in making predictions and interpreting data.

Linear Functions

In mathematics, a function relates one variable to another.

For example, consider the relationship between calorie burn and average pulse.

Generally, as the average pulse changes, so does the calorie burn, making calorie burn dependent on the average pulse.

As the average pulse increases, calorie burn is likely to increase as well.

Here, calorie burn is the dependent variable, and average pulse is the independent variable.

The relationship between these variables can often be expressed using a mathematical formula (function).

A linear function has one independent variable (x) and one dependent variable (y), and is expressed as:

y = f(x) = ax + b

This function calculates the dependent variable’s value based on the chosen value of the independent variable.

Explanation

  • ( f(x) ): The output (dependent variable)
  • ( x ): The input (independent variable)
  • ( a ): Slope, the coefficient of the independent variable, indicating the rate of change of the dependent variable
  • ( b ): Intercept, the value of the dependent variable when ( x = 0 ), and the point where the line crosses the vertical axis

Linear Function with One Explanatory Variable

A function with one explanatory variable uses one variable for prediction.

For instance, to predict calorie burn using average pulse, we use the formula:

f(x) = 2x + 80

Here’s what the components mean:

( f(x) ): The output, predicting Calorie_Burn

( x ): The input, Average_Pulse

( 2 ): Slope, indicating how much Calorie_Burn increases if Average_Pulse increases by one unit

( 80 ): Intercept, a fixed value representing the dependent variable when ( x = 0 )

Plotting a Linear Function

Linearity means a “straight line.”

Graphically, a linear function will always be a straight line, which can slope upwards, downwards, or be horizontal or vertical.

Data Science - Slope and Intercept

Slope and Intercept

Let’s explain how we determine the slope and intercept of our function:

f(x) = 2x + 80

The image below highlights the slope, which indicates the steepness of the line, and the intercept, which is the value of ( y ) when ( x = 0 ) (the point where the line crosses the vertical axis).

The red line continues from the blue line shown previously.

Finding the Slope

The slope shows how much calorie burn increases as the average pulse increases by one unit. It indicates the steepness of the line.

We can find the slope using the proportional difference between two points on the graph:

If the average pulse is 80, the calorie burn is 240.

If the average pulse is 90, the calorie burn is 260.

We see that if the average pulse increases by 10, the calorie burn increases by 20.

Slope = 20⁄10 = 2

Mathematically, the slope is defined as:

Slope = f ( x2 ) − f ( x1 ) x2 − x1 \text{Slope} = \frac{f(x_2) - f(x_1)}{x_2 - x_1}

Where:

  • ( f(x_2) = 260 ) (second observation of Calorie_Burnage)
  • ( f(x_1) = 240 ) (first observation of Calorie_Burnage)
  • ( x_2 = 90 ) (second observation of Average_Pulse)
  • ( x_1 = 80 ) (first observation of Average_Pulse)
Slope = 260 − 240 90 − 80 = 2 \text{Slope} = \frac{260 - 240}{90 - 80} = 2

Ensure the observations are defined in the correct order to maintain accuracy.

Using Python to Find the Slope

Calculate the slope with the following code:

  1. def slope(x1, y1, x2, y2):
  2.         s = (y2 - y1) / (x2 - x1)
  3.         return s
  5. print(slope(80, 240, 90, 260))

Finding the Intercept

The intercept fine-tunes the function’s ability to predict Calorie_Burnage. It is the value of ( y ) when ( x = 0 ).

Here, if the average pulse (( x )) is zero, the calorie burn (( y )) is 80. Thus, the intercept is 80.

While the intercept may not always have a practical meaning (e.g., an average pulse of zero is unrealistic), it is necessary for the mathematical function.

Other examples where the intercept has practical meaning:

  • Predicting next year’s revenue based on marketing expenditure.
  • Fuel usage when a car is idle.
